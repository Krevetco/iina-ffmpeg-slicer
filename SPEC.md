# VideoMarkers — Спецификация плагина

Версия: 0.1.0  
Платформа: macOS, IINA 1.3.0+

---

## Что делает плагин

VideoMarkers — плагин для IINA, который позволяет размечать видео временными метками двух типов прямо во время просмотра, а затем вырезать отмеченные сегменты через ffmpeg.

---

## Функциональные блоки

### 1. Добавление меток

- **Клавиша `1`** — добавляет метку Type 1 (зелёная, назначение: начало сегмента)
- **Клавиша `2`** — добавляет метку Type 2 (красная, назначение: конец сегмента)
- При добавлении: OSD-уведомление с типом и временем (`Type 1 marker — 00:02:58`)
- Время фиксируется в момент нажатия с точностью до секунды

### 2. Боковая панель (вкладка Markers)

#### Информационная строка
- Название текущего видео
- Счётчик меток
- Текущее положение воспроизведения (обновляется каждые 500 мс)

#### Timeline
- Визуальная шкала длительности видео
- Зелёные точки — метки Type 1
- Красные точки — метки Type 2
- Белая вертикальная черта — текущая позиция воспроизведения (обновляется каждые 500 мс, в том числе при паузе и при перемотке через плеер)
- **Клик по timeline** — перемотка к этой позиции

#### Кнопка возврата (появляется после прыжка по метке)
- Появляется под timeline после навигации через плагин
- Текст: `↩ Return to HH:MM:SS`
- Клик — возвращает на позицию до последнего прыжка
- Автоматически исчезает, если пользователь вручную перемотал видео через плеер

#### Два параллельных списка меток
- Левая колонка — Type 1, правая — Type 2
- Метки отсортированы хронологически внутри каждого списка

#### Карточка метки
Каждая карточка содержит:
- Время в формате `HH:MM:SS`
- Label (подпись), если задана
- Кнопку `✏` — редактировать label прямо в карточке (inline, Enter/Esc)
- Кнопку `✖` — удалить метку
- Кнопку `✔` — выбрать метку для пары (работает как радио-кнопка внутри колонки)

**Клик по карточке** (вне кнопок) — перемотка к времени метки.

### 3. Выбор пары и вырезка сегмента

1. Нажать `✔` на карточке из Type 1 → метка выделяется (белая рамка)
2. Нажать `✔` на карточке из Type 2 → появляется модальное окно

**Модальное окно:**
- Если Type 1 стоит раньше Type 2 по времени:
  - Заголовок: "Вырезать сегмент между двумя маркерами?"
  - Информация об обоих маркерах (время + label)
  - Кнопки: **Нет** (закрыть) / **Да** (запустить вырезку)
- Если Type 1 стоит позже Type 2 (неправильный порядок):
  - Сообщение: "Чтобы вырезать сегмент, второй маркер должен идти после первого"
  - Только кнопка **Нет**

**При нажатии «Да»** запускается:
```
ffmpeg -ss {time_type1} -to {time_type2} -i {исходный_файл} -c copy {выходной_файл} -y
```

Выходной файл сохраняется в ту же папку, что и исходный.

**Формат имени выходного файла:**
```
ИмяФайла_ЛейблТипа1_ЛейблТипа2_HH_MM_SS_HH_MM_SS.расширение
```
- Элементы без значения (пустой label) пропускаются
- Русские символы в именах поддерживаются
- Запрещённые символы файловой системы (`/ \ : * ? " < > |`) заменяются на `_`

После успешной вырезки: OSD с именем файла + папка открывается в Finder.

### 4. Экспорт списка меток

Кнопка **⬆ Export** рядом с заголовком Markers:
- Создаёт текстовый файл со всеми метками текущего видео
- Метки упорядочены хронологически (независимо от типа)
- Файл сохраняется в папку данных плагина, затем открывается в Finder

**Формат файла:**
```
00:02:58
Название сцены
Type 1

00:03:45
Type 2
```

**Имя файла:** `НазваниеВидео_YYYY-MM-DD.txt`

### 5. Сохранение данных

- Метки хранятся в IINA Preferences (`iina.preferences`)
- Ключ: `markers_{имя_без_расширения}_{длительность_секунд}`
- Данные загружаются автоматически при открытии видео
- Метки сохраняются при каждом изменении (добавление, удаление, переименование)
- Метки не привязаны к пути файла — только к имени и длительности

## Технический стек

| Компонент | Технология |
|---|---|
| Язык | TypeScript |
| Компилятор | esbuild (IIFE, ES6) |
| Среда | IINA Plugin API (JavaScriptCore) |
| UI | HTML + CSS + Canvas (WKWebView) |
| Хранение | IINA Preferences API |
| Вырезка | ffmpeg (через `utils.exec`) |

### Коммуникация main ↔ sidebar

```
Main → Sidebar:
  sidebar.postMessage('update', { markers, duration, currentTime, videoName })
  sidebar.postMessage('tick', { currentTime })   // каждые 500 мс

Sidebar → Main:
  iina.postMessage('seek', { time })
  iina.postMessage('delete-marker', { id })
  iina.postMessage('rename-marker', { id, label })
  iina.postMessage('cut-segment', { id1, id2 })
  iina.postMessage('export-markers', {})
  iina.postMessage('request-update')
```

### Важный порядок инициализации

`sidebar.onMessage()` должен вызываться **строго после** `sidebar.loadFile()` — `loadFile` сбрасывает все слушатели. Поэтому все обработчики регистрируются внутри хендлера `iina.window-loaded`.

## Ограничения

- Метки идентифицируют видео по имени файла и длительности. Если файл переименован или перекодирован с другой длительностью — метки не найдутся
- ffmpeg должен быть установлен и доступен в `$PATH` (стандартный Homebrew-путь: `/usr/local/bin/ffmpeg`)
- Вырезка использует `-c copy` (без перекодирования) — клип начинается с ближайшего ключевого кадра
- Плагин работает только с локальными файлами (не сетевые потоки)
