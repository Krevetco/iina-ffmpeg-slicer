# VideoMarkers — плагин для IINA

Плагин позволяет ставить временные метки двух типов прямо во время просмотра видео.

## Что умеет

- **Клавиша `1`** — добавить метку Type 1 (зелёная, начало сегмента)
- **Клавиша `2`** — добавить метку Type 2 (красная, конец сегмента)
- **Вкладка Markers** в боковой панели IINA — список меток и визуальная timeline
- **Клик по метке** в списке — переход к этому моменту видео
- **Сохранение** — метки привязаны к конкретному видео и загружаются при повторном открытии
- **OSD уведомление** на экране при добавлении каждой метки

## Требования

- macOS 10.11 или новее
- IINA 1.3.0 или новее (протестировано на 1.4.1)

## Установка (для пользователя)

### Способ 1 — из готового пакета

1. Скачайте файл `VideoMarkers.iinaplgz` из релизов
2. Дважды кликните по файлу — IINA установит плагин автоматически
3. Перезапустите IINA

### Способ 2 — вручную

```bash
unzip VideoMarkers.iinaplgz -d ~/Library/Application\ Support/com.colliderli.iina/plugins/
```

Перезапустите IINA.

### Проверка установки

После перезапуска IINA:
- Откройте **Settings → Plugins** — в списке должен быть **VideoMarkers** (включён)
- Откройте любое видео
- В боковой панели появится вкладка **Markers**
- В меню **Plugin** появятся пункты "Add Type 1 Marker" и "Add Type 2 Marker"

## Что тестировать

### 1. Базовая работа горячих клавиш

- Откройте любое видео
- Начните воспроизведение, перемотайте на середину
- Нажмите **1** → должно появиться OSD уведомление "Type 1 marker — 00:01:23"
- Нажмите **2** → должно появиться OSD уведомление "Type 2 marker — 00:01:45"
- Нажмите ещё несколько раз в разных местах

### 2. Отображение в sidebar

- Откройте боковую панель (View → Show Sidebar или кнопка в интерфейсе)
- Перейдите на вкладку **Markers**
- Должно быть видно: название видео, количество меток, список и timeline
- После нажатия 1/2 список должен обновляться без перезапуска

### 3. Навигация по меткам

- Кликните на любую метку в списке
- Воспроизведение должно перепрыгнуть к этому времени

### 4. Сохранение и загрузка

- Добавьте несколько меток
- Полностью закройте IINA
- Откройте то же видео снова
- Вкладка Markers должна показать ранее добавленные метки

### 5. Разные видео

- Откройте другое видео — метки должны быть пустые (или свои, если уже добавляли)
- Вернитесь к первому видео — метки должны загрузиться обратно

### 6. Консоль для отладки

Откройте **Help → Developer Tools** → вкладка Console.  
При нормальной работе должны быть видны логи:
```
[VideoMarkers] Plugin initialised ✓
[VideoMarkers] iina.window-loaded → loadFile sidebar
[VideoMarkers] iina.file-loaded → videoId: MyVideo_3600
[VideoMarkers] Loaded 0 markers
[VideoMarkers] postMessage update → 0 markers
```
После нажатия **1**:
```
[VideoMarkers] Added type 1 at 00:01:23
[VideoMarkers] postMessage update → 1 markers
```

## Разработка

### Требования

- Node.js 18+
- npm

### Установка зависимостей

```bash
npm install
```

### Dev-режим (watch + автосборка + автоустановка)

```bash
npm run dev
```

Скрипт:
1. Компилирует TypeScript через esbuild
2. Копирует файлы в `~/Library/Application Support/com.colliderli.iina/plugins/VideoMarkers.iinaplugin`
3. Следит за изменениями и повторяет при каждом сохранении

После каждой пересборки — **перезапустите IINA** для применения изменений.

### Разовая сборка

```bash
npm run build
```

### Проверка типов

```bash
npm run typecheck
```

### Сборка пакета для распространения

```bash
npm run package
```

Создаст файл `VideoMarkers.iinaplgz` в корне проекта.

### Структура проекта

```
src/index.ts              — главная логика (горячие клавиши, хранение, события)
ui/sidebar/
  index.html              — разметка sidebar
  style.css               — стили (light + dark theme)
  script.ts               — логика sidebar (рендеринг, навигация)
build/build.mjs           — скрипт сборки на esbuild
Info.json                 — метаданные плагина
dist/                     — скомпилированный плагин (генерируется автоматически)
```

### Как работает коммуникация main ↔ sidebar

**Main → Sidebar** (при добавлении метки или загрузке файла):
```typescript
sidebar.postMessage('update', { markers, duration, currentTime, videoName });
```

**Sidebar → Main** (при клике на метку):
```typescript
iina.postMessage('seek', { time: 123.45 });
```

**Sidebar → Main** (при загрузке sidebar):
```typescript
iina.postMessage('request-update');
```

## Известные ограничения

- Метки привязаны к видео по имени файла + длительности. Если файл переименовать или обрезать — старые метки не найдутся
- Вырезка через ffmpeg использует `-c copy` (без перекодирования) — клип начинается с ближайшего ключевого кадра, небольшой сдвиг нормален
- ffmpeg должен быть установлен (Homebrew: `brew install ffmpeg`)
- Плагин работает только с локальными файлами
